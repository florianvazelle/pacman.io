
# pacman.io

![home](https://florianvazelle.github.io/resources/images/pacman/home.png)  ![game](https://florianvazelle.github.io/resources/images/pacman/game.png)

### Prérequis
 * Nodejs v10.15.0

### Installation
Pour installer les dépendances du projet :
```
 npm install
```
### Démarrage
Pour lancer le serveur :
```
 node server.js
```
